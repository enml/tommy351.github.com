Transmission - 啟動你的 BT 引擎

簡潔快速的 BT 軟體，可完整支援 DHT, PEX, Magnet，甚至內建 Web UI，讓你在外也能下載 BT。

我不喜歡 uTorrent for Mac 的原因主要是因為資源耗太兇了（CPU 佔用率超高），而且介面都是 Windows 的影子，但功能和 WIndows 比起來根本是閹割再閹割，RSS, Magnet 的功能根本就沒有。而毒蛙...Java 寫的軟體就算了吧。

教學

  1. 把下載的壓縮檔解壓縮。
  2. 在 Transmission 按右鍵，選擇「顯示套件內容」。
  3. 將壓縮檔內的zh_TW.iproj放入Content→Resources內。
  4. 重新開啟 Transmission 即可。

或者，也可以直接下載完整檔案。

後記

Transmission 自從踏入 2.0 之後就沒有中文化了（原本的中文化作者怎麼了？），而且效能還變得有些差，直到 2.0.3 之後，差不多回到原本的效能了，才決定自己動手做中文化，改天再去通知官方吧。

話說寒假時一直聯絡 XLD 的作者都沒有回音，反倒是 BetterZip 的作者一聯絡就馬上回應了XD。
