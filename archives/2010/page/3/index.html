BetterZip - Windows式的壓縮軟體

BetterZip 就像是 Windows 上的 WinRAR 或 7zip 一樣，可以直接檢視壓縮檔內容，只解壓縮部分項目。製作方面當然也是一樣，無論是分割還是加密都一應俱全。

可製作的壓縮檔格式：（至於可開啟的壓縮檔格式則是一拖拉庫，這裡就不詳細列出了）

  • ZIP
  • TAR，不壓縮
  • TAR，GZip 壓縮 (TGZ)
  • TAR，BZip2 壓縮 (TBZ2)
  • 7-zip
  • XAR
  • RAR (需安裝外部 rar 元件)

Vox - 羽量級音樂播放器

Vox 是一款輕巧的播放器，麻雀雖小，五臟俱全，支援了許多格式：FLAC, MP3, AAC, Musepack, Monkey's Audio, OGG Vorbis, Apple Lossless, AIFF, WAV, IT, MOD, XM, 遊戲音樂和其他...。此外，可以利用「效果」做出許多不同的變化，並把這些效果連同音樂一同轉換輸出，例如：等化器、殘響、時間延伸、音高移動、回音、以及更多更多的效果等待你發掘（因為我不會翻譯XD，小弟不才，只學過古典鋼琴）！
