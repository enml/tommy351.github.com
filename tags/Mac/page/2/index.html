Paintbrush - 隨手塗鴉小畫家

Paintbrush 是 Mac 上形同於「小畫家」的軟體，可讓使用者快速、簡單的編輯影像，雖然只是個簡單的功能，卻被Mac官方遺棄了許多年。總而言之，本軟體可支援 BMP, PNG, JPEG, TIFF, GIF，也可將圖片複製貼上至一般的Mac應用程式，包含 Microsoft Office和 Apple iWork。
